### **7. Analytics SQL Queries (Ready to Implement)**

#### **Activation Metrics**
```sql
-- Time to first vault entry
SELECT author_id, MIN(timestamp) as first_entry_created
FROM commits
WHERE action = 'Creation' AND entity_type = 'VaultEntry'
GROUP BY author_id;
```

#### **Engagement Metrics**
```sql
-- Sharing frequency
SELECT author_id, COUNT(*) as share_events
FROM commits
WHERE branch = 'milestone/entry_share'
GROUP BY author_id;
```

#### **Compliance Metrics**
```sql
-- Compliance rule validation success rate
SELECT 
  validation->>'checked_by' as rule_template,
  COUNT(*) as total_validations,
  SUM(CASE WHEN validation->>'passed' = 'true' THEN 1 ELSE 0 END) as passed
FROM commits
WHERE timestamp > NOW() - INTERVAL '90 days'
GROUP BY validation->>'checked_by';
```

#### **Anomaly Detection**
```sql
-- Detect bulk access anomaly
SELECT author_id, COUNT(*) as access_count
FROM commits
WHERE action = 'Access' 
  AND timestamp > NOW() - INTERVAL '5 minutes'
GROUP BY author_id
HAVING COUNT(*) > 100;
```
